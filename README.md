# 全球文化晨报（Global Culture Briefing）

一个面向长期阅读的简体中文文化编辑项目：每天用一份高信噪比网页晨报，覆盖 **音乐、电影、剧集、书籍、游戏** 五大领域，帮助读者快速判断今天最值得听、看、读、玩的内容，以及最值得花时间读的行业分析。

在线阅读：<https://salsensei.github.io/culture-briefing/>

## 这是什么

《全球文化晨报》不是“热点搬运”，也不是把几十条链接粗暴堆成一个信息垃圾场。

它更接近一份轻量但专业的文化编辑部日报：

- **全球视角**：优先关注中国境外信息源与国际文化市场动态
- **编辑判断**：不是只贴标题，而是给出筛选、推荐与取舍
- **长期可读**：页面按“出版物”而不是“聊天记录”来设计
- **轻量发布**：用静态网页直接交付，打开即读、分享即看

## 内容结构

每期默认覆盖五个固定板块：

1. **音乐 Music**
2. **电影 Film**
3. **剧集 TV / Series**
4. **书籍 Books**
5. **游戏 Games**

并通常包含三类信息：

- **今日概览**：当天文化消费与行业动态的高层判断
- **新作推荐**：值得优先消费的作品与项目
- **深度专题**：值得进一步阅读的分析、报道或评论

## 为什么做成网站

这个项目选择网页而不是长聊天消息，是因为网页更适合成为“正式发布物”：

- 链接稳定，方便分享
- 阅读体验完整，不被聊天记录打断
- 首页永远指向最新一期
- 最近 10 期归档可快速回看
- 更适合长期积累成一个有辨识度的内容产品

## 适合谁

这份晨报适合：

- 想用 **5–10 分钟** 掌握当天文化世界重点的人
- 对全球文化内容有持续兴趣的人
- 不想被平台算法喂垃圾，希望有人先做筛选的人
- 关注 AI 时代内容消费、发行、评论与文化产业变化的人

## 站点结构

站点保持极简，只保留对读者真正有用的部分：

- `index.html`：最新一期首页
- `archive/index.html`：最近 10 期归档目录
- `archive/YYYY-MM-DD.html`：单期历史页面

## 项目原则

- **质量优先于数量**
- **筛选优先于堆砌**
- **判断优先于复述**
- **长期价值优先于短期热度**
