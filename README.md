# culture-briefing

《全球文化晨报》是一个面向长期阅读的简体中文全球文化编辑项目，覆盖音乐、电影、剧集、书籍、游戏五大领域。

## 它想解决什么问题
不是追热点，也不是搬运热搜，而是每天用尽可能克制、可靠、可追溯的方式，帮读者快速判断：
- 今天哪些文化内容真正值得关注
- 哪些新作值得听、看、读、玩
- 哪些行业变化值得继续追踪

## 这份晨报的价值
- **全球视角**：优先使用中国境外权威来源，避免长期困在单一中文舆论场
- **编辑判断**：不是堆链接，而是先筛选、再归纳、再给判断
- **五大板块完整覆盖**：音乐、电影、剧集、书籍、游戏
- **长期订阅友好**：强调可读性、稳定结构和日更积累价值
- **可追溯**：每条内容都尽量附上明确来源，方便继续深挖

## 目录说明
- `index.html`：最新一期《全球文化晨报》
- `archive/index.html`：历史归档页
- `archive/YYYY-MM-DD.html`：往期晨报页面
